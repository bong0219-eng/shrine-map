# V20 최종 보정

## 고친 문제

### 1. 매일미사·기도문·성가 팝업 뒤로가기
- 팝업에서 뒤로가기 시 앱 종료가 아니라 팝업을 닫고 커버에 남도록 보정했습니다.
- 커버에서는 다시 뒤로가기 1회 토스트, 2회 종료 기준으로 유지합니다.

### 2. 기도문 뒤로가기
- 기도문 본문 → 뒤로가기 → 기도문 목록
- 기도문 목록 → 뒤로가기 → 매일미사·기도문·성가 팝업
- 팝업 → 뒤로가기 → 커버

### 3. 문의·건의 탭 스와이프
- 실제 문의 화면 구조와 관계없이 탭바 후보를 찾아 좌우 스와이프를 바인딩하도록 강화했습니다.
- 탭바 높이/색상/버튼 높이를 관구·교구 화면 느낌으로 맞췄습니다.

## 유지
- Cloudflare Worker REST Key 분리 구조 유지
- 공개 앱 코드에 KAKAO_REST_KEY 없음
