# V20 문의 탭 스와이프 + 기도문 뒤로가기 흐름 수정

## 문의·건의
- 문의·건의 화면 탭을 좌우 스와이프로 이동 가능하게 보정했습니다.
- 탭바 높이/배경/버튼 높이를 관구·교구 화면 느낌에 맞춰 통일했습니다.

## 기도문 뒤로가기
원하는 흐름으로 고정했습니다.

1. 기도문 본문 → 뒤로가기 → 기도문 목록
2. 기도문 목록 → 뒤로가기 → 매일미사·기도문·성가 팝업
3. 팝업 → 뒤로가기 → 커버
4. 커버 → 뒤로가기 1회 토스트, 2회 종료

## 유지
- V20/V20의 Cloudflare Worker REST Key 분리 구조 유지
- 공개 앱 코드에 KAKAO_REST_KEY 없음
